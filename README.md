
# Youtube Music OBS Extension

This is a Google Chrome extension that formats a Youtube Music tab to look better in OBS!

The design is extremely simple, but allows for the .css to be modified to enable more customization. The title scrolls to fit in longer titles

## Setup

1. Download the repository
2. Extract into a folder
3. Go to [Chrome Extensions](chrome://extensions) on your browser
4. Select 'Load unpacked' at the top
5. Select the folder containing the files

## Notes

All this app does is listen to changes on the bottom bar and modifies the elements to display a smaller, clean design as songs change. There is some CSS to clean up the design, but it's nothing special.
